# 🩺 AI-Powered Nail Disease Diagnostic System

## Overview
An intelligent healthcare platform that combines **YOLOv11 Computer Vision** and **Gemini API** to detect nail diseases from images and provide context-aware treatment guidance.

The system analyses uploaded nail images, identifies potential diseases using a trained object detection model, retrieves relevant medical information from a knowledge base, and generates user-friendly explanations and recommendations through Generative AI.

---

## ✨ Key Features

### 🔍 AI-Powered Disease Detection
- Detects multiple nail diseases using a trained YOLOv11 model.
- Real-time image analysis and prediction.
- Visual disease localisation with bounding boxes.

### 🤖 Intelligent Treatment Guidance
- Gemini AI integration for contextual explanations.
- Personalised treatment recommendations.
- Easy-to-understand diagnostic insights.

### 📚 Knowledge Base Retrieval
- Disease-specific information retrieval.
- Evidence-based treatment suggestions.
- Enhanced AI responses using domain knowledge.

### 🔐 Secure User Management
- User registration and login.
- JWT-based authentication and authorisation.
- Protected routes and secure API access.

### 📊 Prediction History
- Store and manage previous predictions.
- Review diagnostic results at any time.

### 🎨 Modern User Interface
- Responsive React frontend.
- Interactive and user-friendly design.
- Seamless image upload and result visualisation.
